用户和角色

按角色阅读的用户手册

English | 中文

第一次使用本手册？请先读操作指南，再回到本页选择与操作问题相匹配的角色。

如果您想按岗位而不是按模块阅读，请从本节开始。每个角色页面都会说明日常流程、需要打开的页面、交接前要检查的内容，以及常见问题应联系谁处理。

操作流程

  1. 当零件、BOM、配方、机台或 NC 程序存在疑问时，先检查工程准备情况。
  2. 使用计划员流程确认工单、排程、释放和队列准备情况。
  3. 使用操作员流程理解如何找到、开始、报工、检验并完成作业。
  4. 使用主管排查流程检查阻塞和交接。
  5. 当检验、NCR、SMARTQC 或校准影响作业时，使用质量流程。
  6. 当访问权限、作业人员身份、角色、翻译或主数据前提影响用户能看到的内容时，使用管理员设置清单。
  7. 当状态或队列术语不明确时，使用操作词汇表。
角色从这里开始主要目标
车间操作员操作员执行已分派作业、上报产出、完成必要检查
生产计划员计划员把需求转成已排程、已释放并可跟踪的工单
生产工程师生产工程师维护路线、配方、机台能力与 NC 程序准备情况
生产主管 / 线长生产主管监控班次、处理工单阻塞、协调交接
质量工程师质量工程师定义检验计划、复查记录、管理 NCR 与校准
系统管理员用户和角色通过可见管理页面维护用户访问
