Role-Based User Manuals

English | zh-CN

New to the workflow? Start with the operating guide, then return here to choose the role that matches your review question.

Use this section when you want a practical manual by job role instead of by module. Each role page lists the normal daily workflow, the screens to open, what to check before handing work to the next person, and where to escalate common problems.

Operating Flow

  1. Check engineering readiness when the part, BOM, recipe, machine, or NC program is in question.
  2. Use the planner walkthrough to review the work order, schedule, release, and queue readiness.
  3. Use the operator walkthrough to understand how work is found, started, reported, inspected, and completed.
  4. Use the supervisor triage to check blockers and handover.
  5. Use the quality flow when inspection, NCR, SMARTQC, or calibration affects the job.
  6. Use the admin setup checklist when access, worker identity, roles, translations, or master-data prerequisites affect what a user can see.
  7. Use the operating glossary when a status or queue term is unclear.
