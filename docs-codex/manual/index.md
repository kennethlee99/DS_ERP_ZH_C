# DS_ERP User Manual

Use this manual by role and task. Start with the line that matches what you need to do today. If a page marks something as `needs-decision`, stop there and collect the requested evidence instead of guessing the click, status, metric, or setup relationship.

| Run the next shop-floor job as an operator | [Operator next-job walkthrough](en/01-workflows/operator-run-next-job.md) | [Operator manual](en/03-by-role/operator.md), [Queue System](en/10-production/queue-system.md), assigned reporting screen | Dispatch rule, row start action, reporting screen, or handover signal is not confirmed. |
| Supervise a shift and triage blockers | [Supervisor triage walkthrough](en/01-workflows/supervisor-triage.md) | [Production supervisor manual](en/03-by-role/production-supervisor.md), [Production Orders](en/10-production/production-orders.md), [Queue System](en/10-production/queue-system.md), [Dashboards](en/10-production/dashboards.md) | A dashboard metric, queue status, or owner is unclear. |
