# 操作术语表

> [English](../en/00-glossary.md) | 中文

当页面使用容易混淆的生产术语时，请先查看本术语表。

| 术语 | 在本手册中的含义 | 下一步检查 |
|---|---|---|
| WO | 工单。计划员和主管复查的生产作业记录。 | 打开[工单](10-production/production-orders.md)。 |
| 作业 | 操作员或主管预期在 Queue System 中看到的工作。 | 将队列行与工单、零件、工序和机台进行匹配。 |
| 队列行 | Queue System 中一条可执行或可复查的可见工作行。 | 检查零件、工序、机台/工作区域和状态。 |
| 已排程 | 作业已经在 Planning 页面放入日期、机台、组别或视图中。 | 对照[计划](10-production/planning.md)和[队列系统](10-production/queue-system.md)。 |
| 已释放 | 作业预期可以进入执行。 | 确认它出现在[队列系统](10-production/queue-system.md)。 |
| 准备好 | 可见状态或条件显示作业可以进入下一步。 | 确认必要设置和检验没有阻塞作业。 |
| 进行中 | 作业看起来正在执行，或被选为当前作业。 | 更改前先检查 Queue System 和相关工单。 |
| 阻塞 | 作业必须等其他角色解决设置、状态、访问或质量问题后才能继续。 | 使用对应角色的升级表。 |
| 完成 | 作业或检验在相关页面显示已保存的完成状态。 | 检查队列行、工单状态或检验记录。 |
| 检验触发 | 可见信号显示继续作业前需要质量录入或复查。 | 打开[SMARTQC 检验](35-smartqc/inspection-data-entry.md)或[检验记录](30-quality/inspection-records.md)。 |
| 用户 | 用于登录网页应用的账号。 | 检查[用户和角色](40-administration/users-and-roles.md)。 |
| 工人 | 如果现场流程使用，则表示车间身份或分派记录。 | 与负责人确认用户与工人的关系。 |
| 角色 | 分配给用户的可见访问分组。 | 将角色与侧边栏和可见操作对照。 |
| 权限标签 | Roles 页面上可见的标签，可能解释菜单或操作访问。 | 除非负责人确认，不要把它解释成技术规则。 |
| 可见操作 | 登录用户能看到的按钮、图标、行操作、菜单项或弹窗操作。 | 如果没有标签，请先请求截图再写明确点击步骤。 |

## 术语仍不明确时

不要猜测。记录您正在使用的页面、行、状态和操作，然后联系流程页面中指定的角色处理。
